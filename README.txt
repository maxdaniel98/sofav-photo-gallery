SOFAV Photo Gallery is a simple plugin to create a photo gallery.

== Description ==
SOFAV Photo Gallery is a WordPress plugin to create a photo gallery, upload photos and manage them in categories.

== Installation ==
Download the plugin .zip file
Go to Plugins > Add New > Upload Plugin
Upload the .zip file
